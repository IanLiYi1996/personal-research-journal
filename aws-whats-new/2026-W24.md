## Summary

这两周 AWS 最大的主题是 **Bedrock 上的前沿模型军备竞赛 + Agent 基础设施进入"运行时"阶段**：

1. **前沿模型集体登陆 Bedrock**：**Claude Fable 5（首个 GA 的 Mythos-class 模型）**、OpenAI **GPT-5.5 / GPT-5.4 / Codex** GA、**Gemma 4**——一周内三大家最新模型齐聚 Bedrock，并新增 OpenAI/Anthropic 兼容 API 的专用控制台与 CloudWatch 指标。
2. **Agent 从"构建"走向"运行与编排"**：AgentCore Runtime 推出**交互式 shell**（可终端接入 agent 会话调试）、Step Functions 新增 **AgentCore 驱动的 agentic reasoning step**（把 agent 推理嵌进工作流）、CLI 推出 **Agent Toolkit**。
3. **SageMaker 训练范式升级**：**多轮强化学习（multi-turn RL）做 agent 模型定制**、Studio 秒级启动且开箱即可定制、HyperPod 加 EFA-only 网络接口与 AI 编码助手排障技能。
4. **Compute 新硬件密集发布**：EC2 **M9g/M9gd（新一代 Graviton）**、**P6-B200 进 GovCloud**、metal-48xl/96xl、ECS Managed Instances 支持 **Trainium/Inferentia**、Lambda Managed Instances 扩区。
5. **数据库版本大年**：Aurora **PostgreSQL 18**、RDS **PostgreSQL 19 Beta**、Aurora DSQL 支持 **JSONB（带压缩）**。

## Top 5 Highlights

1. **Claude Fable 5 在 Bedrock GA（首个 Mythos-class 模型）** — Mythos 级能力位于 Opus 之上，连同同周登陆的 OpenAI GPT-5.5/5.4/Codex、Gemma 4，Bedrock 成为前沿模型的统一接入面。
2. **Step Functions × AgentCore agentic reasoning step** — 把 agent 推理变成工作流里的一个标准步骤，agent 编排从"自建循环"走向"托管原语"。
3. **AgentCore Runtime 交互式 shell** — 可直接终端接入运行中的 agent 会话做调试/运维，标志 agent 基础设施进入"可观测、可运维"的运行时阶段。
4. **SageMaker 多轮强化学习做 agent 模型定制** — 原生支持 multi-turn RL 训练 agent 模型，把前沿训练范式（与本周 HF 论文的 RLVR/agent 自进化主题呼应）产品化。
5. **EC2 M9g/M9gd + ECS Managed Instances 支持 Trainium/Inferentia** — 新一代 Graviton 实例 + 容器原生跑自研 AI 芯片，自研硅在算力栈两端同步推进。

> **安全提醒**（与本仓库 AWS 安全约束相关）：本期多项 agent 运行时能力（AgentCore Runtime 交互式 shell、CLI Agent Toolkit、Quick 的 MCP VPC 连通性）涉及对外暴露面，落地时应走 VPC 私有连通 + IAM 最小权限，避免任何 `Principal: "*"` 或公网 AuthType: NONE 暴露。
