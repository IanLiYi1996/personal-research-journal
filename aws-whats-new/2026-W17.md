- **Date:** 2026-04-24

- **Source:** https://aws.amazon.com/about-aws/whats-new/recent/feed/

## Summary

本周 AWS 共发布 72 项更新。核心主题：

1. **SageMaker 密集更新**（10/14 AI/ML 公告）：HyperPod 拓扑管理 / 弹性实例组 / 深度健康检查，Unified Studio VPC / 多代码空间 / IdC 域支持，Qwen3.5 模型微调
2. **Lambda Durable Functions 全球扩展**：16 个新 Region + Java SDK GA，AWS 正式推进长时运行 Serverless 工作流
3. **Lambda + S3 Files**：Lambda 可直接挂载 S3 为文件系统，Compute+Storage 融合里程碑
4. **Bedrock AgentCore 新功能**：Managed Harness (preview)、CLI、coding assistant skills
5. **Aurora Serverless 智能扩缩**：性能提升 30%，支持缩至零，适配 Agentic AI 突发负载

## Top 5 Highlights

1. **Bedrock AgentCore Managed Harness** — Agent 开发从「拼装」走向「托管」，model + system prompt + tools 即可定义 Agent
2. **Lambda S3 Files** — Serverless 函数直接挂载 S3 为文件系统，Compute/Storage 边界进一步模糊
3. **Aurora Serverless 智能扩缩 + 缩至零** — 30% 性能提升 + 理解工作负载模式 + 可缩至零，明确定位 Agentic AI 场景
4. **Athena 托管联邦查询** — 12 种数据源无需部署连接器，联邦查询门槛大幅降低
5. **SageMaker HyperPod 三连更新** — 自动拓扑 + 弹性实例组 + 按需健康检查，分布式训练体验全面升级
